HOW TO USE
═══════════════════════════════════════════════════════════════════════════════

1. BUILD GODOT
   Windows: python scripts/master_build.py --windows-only
   Linux:   python scripts/master_build.py --linux-only
   Remote:  python scripts/master_build.py --linux-only --push-github

2. RUN WITH MCP SERVER
   python mcp_server/godot_mcp_launcher.py build/godot_windows.exe \
     --project /path/to/project \
     --port 6005

3. CONNECT MCP CLIENT
   python mcp_client_test.py --host localhost --port 6005 --test

4. CALL TOOLS
   Client sends JSON-RPC requests to localhost:6005
   Tools return results for preview or execute actions

5. PUSH TO GITHUB
   git remote add origin https://github.com/USERNAME/agentic-godot.git
   git push -u origin main
