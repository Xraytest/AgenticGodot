# Blender 3D Reconstruction Skill

## Description
视觉 3D 还原流程化操作技能，实现图像->分析->行动->复盘的循环步骤，支持 15000 次迭代编辑操作，用于从 2D 图像重建 3D 人物模型。

## Usage
```
/blender-reconstruct <image_path> [max_steps=15000]
```

## Workflow
1. **图像分析**：读取并分析目标图像，提取人物特征、服装细节、姿态参数
2. **基础构建**：创建骨骼系统和基础网格
3. **细节雕刻**：逐层添加面部、身体、服装、发型细节
4. **材质纹理**：应用皮肤、服装、头发材质
5. **绑定权重**：骨骼绑定和权重绘制
6. **动画姿态**：匹配原始姿态
7. **光照环境**：设置光照和环境

## Loop Structure
每轮迭代包含：
- **观察**：截图当前 Blender 状态
- **分析**：评估当前进度和下一步操作
- **行动**：执行具体的建模操作
- **复盘**：记录操作结果和调整策略

## Output
- 3D 模型文件 (.blend)
- 操作日志 (包含 15000 次迭代记录)
- 中间状态截图
